# Preferences

Echo includes a small set of preferences so the app can fit your local workflow and visual environment.

<h2 id="language">Language</h2>

Switch the interface language from preferences when you want Echo to match your preferred working language.

<h2 id="theme">Theme</h2>

Use the theme preference to switch between light and dark presentation based on your desktop environment or personal preference.

<h2 id="display">Display</h2>

Adjust display and font preferences to make dense Docker information easier to scan during repeated use.

These settings are intended to keep the app comfortable for long-running local development sessions.
