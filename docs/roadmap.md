# Roadmap

Echo is under active development. The current release focuses on local Docker workflows, while future releases will improve packaging and broaden day-to-day resource management.

<h2 id="packaging">Packaging</h2>

Future releases are planned to include smoother installation flows, signed builds, and notarized macOS packages so first launch requires fewer manual security steps.

<h2 id="resource-workflows">Resource workflows</h2>

Planned improvements include deeper network and resource inspection, plus more import and export workflows for images and volumes.

<h2 id="future-connection-targets">Future connection targets</h2>

Echo currently focuses on local Docker usage. Additional Docker-compatible connection targets are planned for future versions as the product matures.
